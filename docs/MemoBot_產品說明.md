## 1. 產品簡介

### 1.1 產品概述

「MemoBot（咩猛）」是一個 LINE Bot 智能行程助理，讓您透過自然語言對話來管理行程。不需要學習複雜操作，只要像和朋友聊天一樣，就能輕鬆掌控生活與工作。

**核心理念**：「用說的就能管理行程」

### 1.2 目標使用者

本系統適用於：
- 個人工作者與自由接案者
- 業務人員與專案經理
- 需要整合行程管理的一般使用者
- 希望提升工作效率的專業人士

### 1.3 系統特色

| 特色 | 說明 |
|------|------|
| 自然語言輸入 | 用日常中文描述行程，AI 自動理解並建立 |
| Google 同步 | 與 Google Calendar 雙向即時同步 |
| 智能識別 | 自動識別時間、地點、參與者資訊 |
| 衝突檢測 | 自動偵測時間衝突，避免重複預約 |
| 隨時可用 | 在 LINE 中即可使用，無需額外安裝 APP |

## 2. 功能說明

### 2.1 行程管理

#### 2.1.1 自然語言建立行程

用日常語言描述行程，系統自動識別並建立：

| 輸入範例 | 系統識別 |
|---------|---------|
| 「明天下午 3 點開會」 | 日期：明天、時間：15:00、標題：開會 |
| 「下週一早上 9 點和王經理在咖啡廳」 | 日期：下週一、時間：09:00、參與者：王經理、地點：咖啡廳 |
| 「每週五下午 2 點 team meeting」 | 重複：每週五、時間：14:00、標題：team meeting |

支援的時間表述：
- 相對時間：明天、後天、下週一、這週五
- 絕對時間：12/25、1月15日、2025/01/20
- 時間段：早上、中午、下午、晚上
- 精確時間：下午 3 點、15:00、3:30

#### 2.1.2 行程查詢

系統支援多種查詢方式：

| 查詢方式 | 範例 |
|---------|------|
| 時間範圍 | 「這週有什麼行程」「明天的行程」 |
| 關鍵字 | 「有關專案的會議」「和小美的約會」 |
| 時間空檔 | 「這週三到五的上午有哪些時段有空」 |

#### 2.1.3 行程修改與刪除

| 操作 | 範例 |
|------|------|
| 修改時間 | 「把明天的會議改到後天」 |
| 修改地點 | 「會議地點改成線上」 |
| 取消行程 | 「取消明天的午餐約會」 |

#### 2.1.4 Google Calendar 同步

系統與 Google Calendar 雙向同步：

| 方向 | 說明 |
|------|------|
| LINE → Google | 在 LINE 建立的行程自動同步至 Google Calendar |
| Google → LINE | 在 Google Calendar 修改的行程會通知您 |

同步狀態說明：

| 狀態 | 說明 |
|------|------|
| SYNCED | 已與 Google Calendar 同步 |
| LOCAL_ONLY | 僅存在本地，待同步 |
| MODIFIED | 已修改，待重新同步 |

#### 2.1.5 衝突檢測

當新行程與現有行程時間重疊時，系統會自動提醒：

```
⚠️ 時間衝突提醒
您在 12/25 下午 2:00-3:00 已有「部門會議」
是否仍要建立「客戶拜訪」？
```

### 2.2 Google Meet 整合

建立線上會議時自動產生 Google Meet 連結：

```
輸入：「明天下午 3 點線上會議」
結果：系統建立行程並自動產生 Google Meet 連結
```

### 2.3 新手引導

首次使用時，系統會引導您：

1. 歡迎訊息與功能介紹
2. Google 帳號授權流程
3. 基本功能教學
4. 進階功能說明

## 3. 操作流程

### 3.1 首次使用流程

```
1. 在 LINE 中加入 MemoBot 好友
2. 發送任意訊息開始
3. 依照引導完成 Google 帳號授權
4. 開始使用行程管理功能
```

### 3.2 日常使用流程

#### 建立行程

```
您：明天下午 3 點和小明開會
MemoBot：✅ 已建立行程
  📅 2025/12/21（週六）15:00-16:00
  📝 和小明開會
  已同步至 Google Calendar
```

#### 查詢行程

```
您：這週有什麼行程
MemoBot：📅 本週行程（12/16-12/22）

  週一 12/16
  • 09:00 週會
  • 14:00 客戶拜訪

  週三 12/18
  • 10:00 專案會議

  ...
```

#### 查詢時間空檔

```
您：這週三到五下午有空嗎
MemoBot：📅 空檔時段

  週三 12/18
  • 13:00-14:00 ✓
  • 15:00-18:00 ✓

  週四 12/19
  • 13:00-18:00 ✓

  週五 12/20
  • 14:00-16:00 ✓
```

## 5. 安裝說明

### 5.1 加入好友

1. 開啟 LINE APP
2. 點選「加入好友」
3. 搜尋「MemoBot」或掃描 QR Code
4. 點選「加入」

### 5.2 Google 帳號授權

首次使用時需完成 Google 帳號授權：

1. 發送任意訊息給 MemoBot
2. 系統會發送授權連結
3. 點選連結並登入 Google 帳號
4. 同意授權存取 Google Calendar
5. 授權完成，開始使用

## 6. 常見問題 (FAQ)

### Q1: 如何建立行程？

直接用中文描述您的行程即可，例如：
- 「明天下午 3 點開會」
- 「下週一 9 點和王經理見面」

系統會自動識別時間、地點、參與者並建立行程。

### Q2: 為什麼要授權 Google 帳號？

授權 Google 帳號後，您在 LINE 中建立的行程會自動同步至 Google Calendar，讓您在任何裝置都能查看行程。

### Q3: 我的資料安全嗎？

是的，我們非常重視資料安全：
- 所有資料傳輸使用 HTTPS 加密
- 資料儲存於雲端伺服器，定期備份
- 僅存取必要的 Google Calendar 資料
- 不會分享或出售您的個人資料

### Q4: MemoBot 是免費的嗎？

是的，MemoBot 目前完全免費使用，包含所有行程管理功能與 Google Calendar 同步。

### Q5: 系統無法理解我的意思怎麼辦？

請試著用更明確的方式描述，例如：
- ❌ 「下午有事」
- ✓ 「下午 3 點有會議」

如果問題持續，請發送「幫助」查看使用說明。

### Q6: 如何刪除我的帳號和資料？

發送「刪除帳號」，系統會引導您完成帳號刪除流程。請注意，刪除後所有資料將無法復原。

### Q7: Google Calendar 事件沒有同步怎麼辦？

請確認：
1. 已完成 Google 帳號授權
2. 網路連線正常
3. 發送「同步狀態」檢查同步狀態

如問題持續，請發送「重新授權」重新進行授權流程。

### Q8: 支援哪些語言？

目前支援繁體中文。未來計劃支援英文和其他語言。
